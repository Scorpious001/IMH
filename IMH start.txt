To start the backend server:
1. Open PowerShell
2. Navigate to: cd "C:\Users\Steve\Desktop\ScorpiousProductionStudios\IMH\IMH\backend"
3. Run: .\venv\Scripts\python.exe manage.py runserver

OR use the script:
.\start-backend.ps1

To start the frontend:
1. Open a NEW PowerShell window
2. Navigate to: cd "C:\Users\Steve\Desktop\ScorpiousProductionStudios\IMH\IMH\frontend"
3. Run: npm start

OR use the script:
.\start-frontend.ps1

To start both at once:
.\start-all.ps1
